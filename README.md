# 🚀 Docker TestApp (DevOps Project)

## 📌 Overview
This project is a containerized full-stack application built using Docker.

It demonstrates how to package an application along with its database into containers and run it easily on any system.

The system consists of:

- 🖥️ Backend Application (Node.js)
- 🗄️ Database (MongoDB)

---

## 🏗️ Architecture
This project uses a container-based setup.

Services:

- testapp → Main application
- mongo → Database service
